## ⚡ What is Cricket War Room?

**Cricket War Room** is a real-time, AI-powered fan battle arena built for the **Google Agentic Premier League Hackathon**. Fans from rival cricket teams (RCB 🔴 vs DC 🔵) join live lobbies, watch a simulated match unfold ball-by-ball, place micro-bets, receive AI-generated roasts after key moments, and compete to become the most prophetic fan — the **Cricket Oracle** 👑.

No passive watching. No static scoreboards. Pure fan warfare.

---

## 🎥 Demo

```
1. Join a team faction (RCB 🔴 or DC 🔵)
2. Watch the live 3D cricket field update ball-by-ball
3. Place micro-bets on over outcomes before the timer runs out
4. Receive AI-generated roasts when wickets fall or sixes fly
5. Climb the leaderboard by predicting correctly
6. Watch the Oracle Reveal at match end — who was most prophetic?
```

## 🤖 Multi-Agent Architecture

The app runs a **5-agent sequential pipeline** inspired by Google ADK:

```
┌─────────────────────────────────────────────────────────┐
│                   AGENT PIPELINE                        │
│                                                         │
│  [Agent 1]          [Agent 2]          [Agent 3]        │
│  Match              Bet                Settlement        │
│  Simulator  ──────► Generator  ──────► Agent            │
│  (Ball by           (3 dynamic         (Resolves bets,  │
│   ball feed)         bets/over)         awards coins)   │
│       │                                     │           │
│       ▼                                     ▼           │
│  [Agent 4]                           [Agent 5]          │
│  Roast Agent                         Oracle Tracker     │
│  (Claude API —                       (Ranks fans by     │
│   spicy banter                        prediction        │
│   on wickets/sixes)                   accuracy)         │
└─────────────────────────────────────────────────────────┘
```

## 🎮 Features

### 🏟️ Live 3D Cricket Field
- Full stadium with mowing stripe grass, boundary rope, and sandy pitch
- 11 humanoid fielders with idle animations and position labels
- **Real-time ball trajectory** — watch the cricket ball arc over the field when a Six or Four is hit
- Dynamic shadow follows the ball to show depth

### 💰 Micro-Bet Arena
- 3 unique bets generated by **Agent 2** every over
- Countdown timer — bets lock when the over starts
- Live coin balance that updates in real-time via Socket.IO
- Auto-recovery: if the server restarts, your session restores automatically

### 🔥 AI Roast Engine (Agent 4)
- Powered by **Anthropic Claude API**
- Generates spicy, PG-13, team-specific banter after every wicket or six
- Toast notifications fly in over the 3D field

### 📡 Social Pulse (Twitter/IG Style)
- Real-time fan activity feed with DiceBear SVG avatars
- **#TRENDING**, **FAN BETS**, and **ROASTS** tabs
- Click 🔥 ROAST to broadcast banter to all connected fans instantly

### 👑 Cricket Oracle Reveal
- Cinematic end-of-match screen with 3D rotating trophy
- Ranks fans by prediction accuracy + coins earned
- Reveals the secret Traitor (1 in 10 chance per player)

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/cricket-war-room.git
cd cricket-war-room

# Install all dependencies
npm install
cd backend && npm install && cd ..
cd frontend && npm install && cd ..

# Add your Anthropic API key
echo "ANTHROPIC_API_KEY=your_key_here" > backend/.env

# Launch everything with one command
npm start
```

Then open:
- **App:** http://localhost:5173
- The match simulator starts automatically — no admin panel needed!
